# ci-server-prerequisites

#### Prerequisites 

Any command line tools that Continuum needs must be installed on the system and be available in _PATH_. See the exported _PATH_ in [continuum-smf.xml](continuum-smf-xml.md). 

###### Subversion, Maven2 and Ant

```
pkg-get install subversion maven2 ant 
```
