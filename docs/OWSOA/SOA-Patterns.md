### Intro

This page is a collection of some of the key patterns which have proved useful in building Service Oriented Architectures.

### Misc Supporting Patterns

[Command](http://en.wikipedia.org/wiki/Command_pattern) creates objects which encapsulate actions and parameters.  
[Specification](http://en.wikipedia.org/wiki/Specification_pattern) separates the statement of what kind of objects can be selected from the object that does the selection.
