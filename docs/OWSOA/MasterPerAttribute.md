# MasterPerAttribute

This concept applies to mastering strategies in a Core Service with multiple datasources.[excerpt} The concept describes the ability to control mastering on a per-attribute basis, rather than on a per-system basis. Implementing support for [MasterPerAttribute](excerpt-The-concept-describes-the-ability-to-control-mastering-on-a-per-attribute-basis-rather-than-on-a-per-system-basis-Implementing-support-for-MasterPerAttribute.md) delivers a fine-grained control over enterprise data.
