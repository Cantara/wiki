# Build Android projects with Maven

Dev env requirements:
1. Follow http://code.google.com/p/maven<sub>~android</sub>~plugin/wiki/GettingStarted
1. Install platforms:
```bash
android update sdk --no-ui --obsolete --force
```
1. Install 32<sub>~bit libs if you use 64</sub>~bit OS:
```bash
sudo apt-get install ia32-libs
```
