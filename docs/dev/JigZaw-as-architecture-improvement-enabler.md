# JigZaw as architecture improvement enabler

JigZaw hjelper deg til å 
a) se, finne og løse opp i den selvpåførte systemkompleksiteten man har mellom moduler og med å la prosjektet arbeide fritt på tvers av systemmoduler  

b) en oppskrift/løsning som gjør at ryddig modulsentrert arbeid både øker kvalitet og produktivitet i prosjektene
