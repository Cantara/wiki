# Web Design

This page needs ALOT of work, but unfortunately this kind of knowledge is spread evenly throughout the web. Web designers are perhaps the most intensive bloggers out there, so any search on google will provide enourmous bundles of information on any web design topic. This page attempts to refine the most valuable resources.

As programmers, we tend to ignore the aspect of our web applications which is perhaps the most important one: **web design**. Think..

- Images, logos
- Colors
- Fonts
- Layout
- Templates
- User-friendliness
- [POSH](http://microformats.org/wiki/POSH)
- AJAX, [JavaScript](JavaScript.md)
- Internationalization and localization
- Many more...

Some of these topics borders on traditional programmer tasks. It's important to build a solid bridge across these two domains, focusing on..

- cross-knowledge between programmers and designers
