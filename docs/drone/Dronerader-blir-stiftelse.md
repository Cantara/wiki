# Dronerader blir stiftelse

### [Dronerader blir stiftelse](Dronerader-blir-stiftelse.md)

_Regler for stiftelsen her_

Grunneggende skal systemet være transparent og åpent. Alle får samme mulighet for utenforliggende systemer på forretningsmessig grunnlag.

Core system blir definert som:

- Tracker interfacing
- Posisjonsdatabase
- Tracker unit og brukerkontodatabase
- Map og posistionview komponenter

Utenforliggende systemer er definert som:

- Leveranse av trackere
- 3 parts posisjons systemer som leverer data inn i core
- Website som embedder map posisjonsview
- Gameification basert på tracker data
- Gadgets
- Andre støttesystemer

Det betales en avgift til stiftelsen som går til drifting og utvikling av core system. (avtaler senere)
