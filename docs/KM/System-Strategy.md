# System Strategy

You need to select one of these strategies:
1. In-House development silo application.
1. Trade-specific standard application.
1. Supplier-strategy: A single supplier (eg IBM, SAP) solves all problems.
1. Integration-strategy: A selected few application are integrated. Eg DB-synchronization or ETL.
1. Services-strategy: Establish services that supports your business's core processes. (eg. Revenue, HR, Sales, Customers)
