## Kort om [Search-driven Business Intelligence](/web/20200813004217/https://wiki.cantara.no/display/KM/Search-driven+Business+Intelligence "Search-driven Business Intelligence")

### Key features

- 70% of key data is unstructured.
- Real-time data
- Real-time alerts/action
- Good support for unstructured data
- file formats
- Rich correlation

### Schematic

How to do data analysis when the amount of unstructured data just increases?

Search-driven technology: Real-time data aquisition (indexing).
