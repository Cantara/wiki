**Mobile Code Camp 2011** - Noen korte  stikkord :)

| Tid/Sted: | slutten av oktober/første helg i november - en helg. Fra 1800 fredag til 20:00 søndag (litt a-la HTML5 Oslo som du finner på wiki.cantara.no) |
| Deltakere: | ca 50 stk totalt, erfaringt med mobilutvikling en forutsetning. Deltakerkostnad: 2500-3500 (estimert, kostpris) |
| Konsept: | "The Mobile Code Camp will be an exciting hands<sub>~on weekend of mobile technology development. The Code Camp will split the group into several hard</sub>~core teams each given the task of developing a mobile solution to an typical web 2.0 site. We will host both cross-platform and native teams, and we will learn ins and outs of all the current mobile platform alternatives." |
| Målsetning: | Styrke kompetansen (dybde og bredde) i teknologiene og alternativene (platform/teknologi) for spydspissene i norsk mobilutvikling |

**Stikkord:**
- prosjektorientert gjennomføring
- felles "case", med "kunderepresentant" on-site tilgjengelig for teamene
- All dokumentasjon/kode: CC 3.0 Attribution  (republiserbart)
- parallellt presentasjonsspor (opt-in) som belyser forskjellige nøkkelteknologier, trender og tilknyttende områder
- både in<sub>~hyrede "hot</sub>~shots" og god tid til "unconference" med innhold fra deltakere
- HTML5 (med og uten rammeverk som Phonegap), Android og iOS teams
- "ekspert" facillitator for hver teknologi, tilgjengelig for teamene

**Partnerskap:**
- Plan: typisk et lite sett (en håndfull), nivå: ca 20k + MVA
- Innhold
    - branding av web
    - tilgang til informajson
    - on-site banner el. lign.
    - 2-3 reserverte plasser (ressursene må kvalifiseres mhhp kompetanse)
    - give-aways
    - (event skreddersøm ved behov)
