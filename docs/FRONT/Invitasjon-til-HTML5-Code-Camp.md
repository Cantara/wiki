| Invitasjon til HTML 5 code camp **Bli med å dra norsk front end utvikling ut av gjørma**, slik at vi slutter å kaste bort store summer på å lage dårlige web applikasjoner!  HTML5 har fått mye fokus det siste året. For hver dag som går får browsere bedre og bedre støtte for disse nye spesifikasjonene. Både i form av de nye JavaScript-APIene og de nye elementene i HTML5 spesifikasjonen. Dette legger et solid grunnlag for å bygge applikasjoner basert kun på åpne web standarder.  Vi vil gjøre et **prosjektorientert "hands on" dypdykk i HTML 5** over en hel helg. De mest relevante spesifikasjonene vil bli gjennomgått og praktisert ved løsning av realistiske problemorienterte oppgaver.  **Hvem er dette for:**   - Du kjenner godt til HTML og JavaScript, men du har ikke hatt tid til å leke med HTML 5 enda. - Du brenner for å lage bedre løsninger basert på åpne web standarder som kjører på både desktop og mobil.   **HTML 5 code camp** arrangeres av Framsia og Cantara med hensikt å ta tak i utfordringene og mytene rundt web anno 2010.
