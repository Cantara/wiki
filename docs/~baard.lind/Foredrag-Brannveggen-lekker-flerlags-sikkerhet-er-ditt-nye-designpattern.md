# Alternatives:

- Brannveggen lekker - flerlags sikkerhet er ditt nye designpattern.
- En lags sikkerhet døde på 90 tallet, slik sikrer du deg i 2917.
- Sikre nett finnes ikke - flere lags sikkerhet må bli standarden.
- Sikre tjenestene dine - bruk fler lags sikkerhet.

- Be specific
  - general topics has more competition.
  - more interesting when talk is more concrete.

- Abstract: - common problems, what will be solved. Informal
  - Description in Program

- Outline: - pitch to Program Commitee
  - Sections, bulletpoints, timing
  - Be concise

- Supporting materials
  - include references, show your skills.

"the work load on program comitee is high"
