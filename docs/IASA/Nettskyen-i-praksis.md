# Nettskyen i praksis

|  |  |
| --- | --- |
| Title | Nettskyen i praksis |
| Speaker(s) | Peter Flem, Capgemini |
| Date | IASA Norway, onsdag 1.sept 2010, 19:00-19:30 |
| Duration | 30 min |

Presentasjonen er om et prosjekt hvor kunden ønsket et CRM system fra Salesforce. Det vil bli tatt opp de erfaringene som man fikk med å implementere et reelt Cloud Computing prosjekt hos en norsk stor kunde.Her vil det bli vist kode fra Eclipse som viser hvordan Apex koden virker i Salesforce (plattformen heter Force). Dette er Salesforces bruk av PaaS tjeneste, som gjør at IT-leverandører kan tune Salesforce applikasjonen i skyen til å bli tilpasset kundens behov. Dette så vi behov for etterhvert som CRM prosjektet nærmet seg de siste sprintene.

### Agenda

- Intro til Salesforce og Force plattformen
- Litt om kundens krav til systemet
- Utfordringene med implementeringen
- Apex kodingen og Dataloader
- Kundens reaksjon på Cloud tjenesten
- Så hva er forskjellen på tradisjonell CRM og Cloud basert CRM?
