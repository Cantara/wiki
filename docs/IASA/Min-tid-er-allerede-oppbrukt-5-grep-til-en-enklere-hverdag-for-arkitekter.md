# Min tid er allerede oppbrukt - 5 grep til en enklere hverdag for arkitekter

For tidlig til juni møtet?

Utdype etter samtaler med Truls, Nils og andre arkitekter som har hatt, eller er i tidsklemma.

1. Stop og pust, start å reflekter over hva du bruker tiden din på.
2. Aksepter at andre kan få nok oversikt til å ta gode beslutninger.
3. Lag deg grep du kan gjøre for å delegere ansvar og oppgaver. Skriv ned disse grepene.
4. Bygg opp dine medarbeidere til å hjelpe dem å ta rette avgjørelser. La denne prosessen ta litt tid.
5. Bygg feedback sløyfer som gir deg informasjon tilbake fra den virkelighet dine medarbeidere oppdager.

Bonus:
Tør å sett ROI på de avgjørelser som skal tas. Det må ikke være mye jobb - ROI mål skal skapes av forretning!
