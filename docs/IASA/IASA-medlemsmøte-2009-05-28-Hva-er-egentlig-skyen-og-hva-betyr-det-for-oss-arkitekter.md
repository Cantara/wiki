# IASA-medlemsmøte 2009-05-28 - Hva er egentlig "skyen", og hva betyr det for oss arkitekter?

### Medlemsmøte

- **Dato:** 28.Mai 
- **Tid:** 18:00-21:00
- **Sted:** Scotsman 
- **Tema:** Hva er egentlig "skyen", og hva betyr det for oss arkitekter?

### Invitasjon
Cloud computing seiler opp som det neste store som skal gjøre verden vår enklere og bedre. Våre gode foredragsholdere har rukket å vinne praktisk erfaring med "skyen", og de vil dykke ned i cloud computing og hva det kan brukes til (og kanskje ikke brukes til).

Skal man tro leverandørene virker det som at "skyen" som konsept synes å ha livets rett, men er det egentlig bare en avløser for SaaS?

Hva vil "skyen" bety for oss i praksis, og hvordan skal vi som arkitekter forholde oss til den? Er dette viktig?

Som sist kjører vi også engasjerte Open Space for å diskutere disse, og deres, temaer omkring skyen.
