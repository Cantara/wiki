# Laws for project architects

2009.06.24 - Stig Lau

### Intro

- Title:  **Laws for project architects**
- Speaker: Stig Lau, Bouvet AS
- Duration: 35 min + Open Space session
- Audience: Software Architects, Tech leads, Solution Architects, Project Leaders, Product Owners and ScrumMasters.

### Short summary

A rundown of the key rules that applies which makes project architects active and positive contributors to IT in large organizations.

### Abstract

Architects have a pretty bad reputation in the industry. This talk will focus on suggesting a set of rules all project architects should apply to, which will increase the value they produce for the project. This we think will change they way we judge architects and thereby put an end to the bad reputation this group has in the industry.
