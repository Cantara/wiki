# Hvordan smidig2010 bruker nettskyen

Smidig2010 skaleres med scrollbar! Jeg forteller kort om hvordan, viser arbeidsflyten og hvordan én-kommandodeploy fasiliteter continuous deployment, en prosess alle bør strebe etter for å redusere risiko og levere forretningsverdi raskt. Det blir og visning og analyse av Smidig2010.no, monitorert av New Relic.
