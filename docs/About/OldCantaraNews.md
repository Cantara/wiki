# OldCantaraNews

**Stiftelsesperioden forlenget**  
Dugnadsånden i de norske fagmiljøene er unik på verdensbasis. Målet er at Norge skal bli anerkjent som verdensledende innen programvareutvikling. Stiftelsen for fremme av programvareutvikling i Norge etableres for å fokusere på dette. [Les mer](../PE/Stiftelsesinvitasjon.md)

**Mobilkonferansen AppWorks**  
Lær og bli med på apps revolusjonen. Her dekkes teknologien, erfaringer fra virkeligheten og vær med på diskusjonen rundt mulighetene. [Les mer](http://www.appworks.no/index.php)

**HTML 5 Code Camp**  
Bli med å dra norsk front end utvikling ut av gjørma, slik at vi slutter å kaste bort store summer på å lage dårlige web applikasjoner! Vi vil gjøre et prosjektorientert "hands on" dypdykk i HTML 5 over en hel helg. De mest relevante spesifikasjonene vil bli gjennomgått og praktisert ved løsning av realistiske problemorienterte oppgaver. [Les mer](../PE/HTML-5-Code-Camp.md)
