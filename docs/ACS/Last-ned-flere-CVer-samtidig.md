# Last ned flere CVer samtidig

1. Bytt til listevisning
1. Søk på personer du ønsker å laste ned CV for, skill mellom flere personer med "OR"
1. Huk av for CV-profilene du ønsker å laste ned
1. Velg format (Word, PDF eller OpenDocument) og trykk "Download selected CVs"
