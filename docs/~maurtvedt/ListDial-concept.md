# ListDial concept

### App description

Fast. Intuitive. Simple. 

ListDial allow you to manage multiple lists of contacts you are going to call. 
Whenever ready to make the calls just push the Callâ€™em button. 

Very useful for sales people on the road, if you make a lot of phone calls from your car, or if you plan calls you are going to make up front. 
The app use txt-files in a dedicated Dropbox folder to store lists, making it is very easy to sync your lists across multiple devices.
