# Oracle XE til tester

XE støtter bl.a ikke bitmap indexes og partisjoner. Hvis vi skulle bruke XE måtte vi ha egne skjemaer for databasetest. Det er ikke ønskelig. Foreslår derfor at tanken om at å bruke XE skrinlegges.
