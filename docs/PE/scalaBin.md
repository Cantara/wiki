# scalaBin

ScalaBin user group are for all programmers interested in the language Scala ([http://www.scala-lang.org](http://www.scala-lang.org/)). The members are interested in scala's hybrid approach to object-orientation and functional programming, actor based programming for concurrent problems and hot to get the most out of existing Java code and new Scala code in real projects. ScalaBin is a subgroup of JavaBin.
