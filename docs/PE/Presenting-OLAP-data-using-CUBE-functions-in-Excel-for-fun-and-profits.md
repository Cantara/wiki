# Presenting OLAP-data using CUBE functions in Excel for fun and profits

### Presenting OLAP-data using CUBE functions in Excel for fun and profits

**Kort beskrivelse:**  
First comes a whirlwind walk-through of one of the least known and undstood Excel functions: the CUBE\*-family. We’ll then dive right into using these against both Powerpivot and Analysis Services sources in order to create amazing, free-form reports/applications that doesn’t look like Pivottables at all! Warning: No AdwentureWorks data involved!

**Format:**  
Foredrag, 45 min.

**Forventninger fra deltagere:**  
Participants should have at least a rudimentary knowledge of Powerpivot and/or Analysis Services and Excel.
