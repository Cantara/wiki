# engasjer deg i Cantara

|

  |

|  | Vi har ambisiøse mål, og trenger alltid gode folk som vil være med å gjøre Norge til et sentrum for programvareutvikling |

## Vi ser alltid etter flere som kan bidra som:

- Deltagere på våre arrangementer
- Foredagsholdere som deler av sin erfaring
- Ildsjeler som vil være med å lage spennende faglige arrangement
- Folk som er flinke til å spre det gode budskap
- Og folk som vil være med å trekke lasset

## Bli med i dag!

- Meld din interesse som [ildsjel i Cantara](/web/20230925123512/https://wiki.cantara.no/display/PE/Ildsjel+i+Cantara "Ildsjel i Cantara")
- Sjekk våre [Aktiviteter](/web/20230925123512/https://wiki.cantara.no/display/PE/Activities "Activities") og registrer deg under som interessert.
