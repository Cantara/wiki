### Praktisk bruk av HTTP

Som webutvikler har vi lenge kunnet jobbe på abstraksjonslag over HTTP. Men trenden nå er at både rammeverk og arkitekturstiler går "back to basic" og bygger direkte på HTTP. Dette krever at vi utviklere forstår mer av HTTP, hvilke mekanismer det bygger på og hvordan det bør brukes.

Stefan forteller oss hva alle webutviklere bør kunne om HTTP for å utvikle stabile, kjappe og sikre løsninger på web.

Stefan Landrø (BEKK)
