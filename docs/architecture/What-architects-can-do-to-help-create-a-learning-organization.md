# What architects can do to help create a learning organization

These techniques are _examples_ on how the architect can help create a learning organisation.

- Lead by example 

- Naming 
    - Shared terminology - DDD 
        - Developers, testers, business
    - Clean Code prinsipper: Fornuftig navngivning
        - Developers, testers 

- Be available to the team(s) 

- Training 
    - Teach communication skills/techniques 

- Personal qualities 
    - How to train yourself
