# Thou shalt never micromanage project decisions

As an organisation's architect it is important to restrict yourself to making decisions that have measurable impact across the whole organisation.

For example, it is not the organisation architect's role to dictate what logging libraray should be used on a project. However, it could be the organisation architect's role to dictate use of a centralised logging service across all applications.

[Architects focus is on the boundaries and interfaces](http://97-things.near-time.net/wiki/Architects%20focus%20is%20on%20the%20boundaries%20and%20interfaces)
