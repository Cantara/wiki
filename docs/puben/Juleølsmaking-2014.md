# Juleølsmaking 2014

Tradisjonen tro, så er det på nytt klart for årets juleølsmaking, som i år er **den 23. i rekken**.  I fjor fikk vi egetbrygget juleøl. Kan vi håpe på at noen stiller med godt brygg i år også?

### Kritisk info

| Tid | Lørdag 15.11 kl. 18:00 -> |
| Sted | [Altran-Inkognitogata 35 | https://www.google.no/maps/place/Inkognitogata+35,+0256+Oslo] |
| Forberedelse | Kjøpe en og ta med av hver av det **norske** juleølet **på polet**  (Typisk 20-25 stk) :) |

### Stemmeregler

**Vekt**
- Probies:  1/2 vekt
- Qualified voters 1/1 vekt
- Failed probies:  0/2 vekt
- Probies kan ettter årets stemmegivning rykke opp, rykke ned eller forbli probie.

**Stemmekrav**
- Minstevolum: 2dl av enhet 
- Ikke smakt på årets juleøl før selve testen.
- Terningkast
- **Ikke lov å endre tidligere avgitt** stemme(r)
    - Bortsett fra den første stemmen under avgivning av andre stemme
- Forsøk på **taktisk** stemmegiving gir automatisk nedrykk til **failed probie**
- 3 påpekninger om useriøs stemmegiving gir automatisk nedrykk til **failed probie**
- Aktiv taktisk **lobbyvirksomhet** gir automatisk nedrykk til **failed probie**

**Med liten skrift**
- Den først ankomne nybegynner får tildelt oppgaven å lage regnearket/formelverket for riktig beregning av vektede resultater.
