### Node documentation

| ID | Comment |
| --- | --- |
| applicationID | The ID for the application |
| applicationName | The user-friendly name of the application |
| applicationSecret | The applications secret |
| applicationurl | The applications url (For redirect(s)) |
| minimumsecuritylevel | What are the application security contraints (DEFCON levels) |
