# Whydah OpenLDAP Docker image

Work in Progress

Documentation for getting started with creating a OpenLDAP Image.

#### Resources

- Better security and minimalistic: <https://registry.hub.docker.com/u/cptactionhank/openldap/dockerfile/>
- Better documentation: <https://registry.hub.docker.com/u/larrycai/openldap/dockerfile/>
- Ansible Config for javaBin done by Hamnis: <https://github.com/javaBin/Whydah-Provisioning/tree/master/roles/openldap>
- Github project of openldap docker build for Whydah by kimcs: <https://github.com/kimcs/whydah-openldap>

#### Getting started

- docker pull cptactionhank/openldap
- docker run -d --name ldap -t cptactionhank/openldap
