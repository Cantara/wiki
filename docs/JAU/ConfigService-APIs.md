# ConfigService APIs

Visual (graph) representation of the data model: [ConfigService provisioned clients data model](/web/20210415090119/https://wiki.cantara.no/display/JAU/ConfigService+provisioned+clients+data+model "ConfigService provisioned clients data model")

We split the API documentation in two

- [ConfigService Client API](/web/20210415090119/https://wiki.cantara.no/display/JAU/ConfigService+Client+API "ConfigService Client API")
- [ConfigService Admin API](/web/20210415090119/https://wiki.cantara.no/display/JAU/ConfigService+Admin+API "ConfigService Admin API")

The Client API is used by [Java Auto-Update](/web/20210415090119/https://wiki.cantara.no/display/JAU/Java+Auto-Update "Java Auto-Update") managed applications and provide software and configuration for the application they manage.  
The Admin API's are for configuration and setup and visualization of the set of applications a given ConfigService instance control.
